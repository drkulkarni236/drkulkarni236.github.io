Overwiew

This project looks at the publicly available San Francisco bike share data to analyze bike share usage trends and provide recommendations to increase the user base for bike sharing. This project is completed entirely using the google cloud platform (GCP). The data was queried using SQL through big query and through the command line. Jupyter notebook is used to plot the data to generate insights.
